# Извлечение данных из пакета запросов

## Карта запросов: какой ответ что закрывает

| Орган | Что даёт | На что смотреть |
|---|---|---|
| **ЕГРН / Росреестр (Роскадастр)** | Недвижимость: текущая и история прав за период | Даты регистрации И ПРЕКРАЩЕНИЯ прав (сделки!), кадастровые номера, основания (ДКП/дарение/наследство/приватизация), обременения. «Уведомление об отсутствии сведений» = чисто |
| **ЦТДиКО / БТИ** (ГБУ «Центр техдокументации и кадастровой оценки») | Недвижимость, учтённая ДО 31.01.1998 (дореестровая) | Закрывает «историческую» недвижимость, которую не видит ЕГРН |
| **ГАИ / ГИБДД (МРЭО)** | Транспорт: текущий + регдействия за период | Постановка/снятие с учёта, «в связи с продажей», «по заявлению владельца» (= авто могло уйти без ДКП!), светокопии ДКП по запросу. Стандартная приписка: запреты на регдействия — только через суд (ч. 7 ст. 213.9) |
| **Гостехнадзор** (Минсельхоз региона) | Самоходные машины, тракторы, прицепы | обычно «не зарегистрировано с 01.02.2020» |
| **МЧС / ГИМС (отдел БВО)** | Маломерные суда | часто веерный ответ по списку должников |
| **Росавиация** (Южное МТУ) | Воздушные суда «когда-либо» | веерный ответ по списку |
| **ФНС / УФНС (ДСП)** | Счета (форма 67ф/9ф), 2-НДФЛ/3-НДФЛ, ЕГРИП/ЕГРЮЛ, объекты налогообложения, задолженность | Список ВСЕХ счетов (для таблицы блокировки), доходы за 3 года, статус ИП, налоговые объекты — кросс-проверка ГАИ/ЕГРН (ФНС иногда видит ТС, которые ГАИ не показала!) |
| **СФР (ОСФР)** | Пенсии, пособия, факты работы (страхователи) | Вид и размер пенсии, ЕДВ, пособия на детей (ЕГИССО), периоды работы. Если ответа нет — заменяется справкой МСЭ/ПФР от должника |
| **ЗАГС** (по определению суда) | Записи актов: браки, разводы, дети, смерть, перемена имени | Семейное положение, несовершеннолетние дети (иждивенцы!), даты расторжения брака (развод перед банкротством — типичен), смерть супруга. Приложение обычно 10–14 л. копий записей |
| **ЦЗН** | Статус безработного, пособия | закрывает «скрытые доходы» |
| **ФИПС / Роспатент** | Интеллектуальная собственность | обычно пусто |
| **Росгвардия (ЦЛРР)** | Оружие | часто не отвечает — фиксировать |
| **РСА / НСИС** | Страховая история (ОСАГО = косвенный признак авто) | часто не отвечает |
| **УФССП** | Исполнительные производства | часто не отвечает |
| **МВД (ОВМ)** | Адресная справка, паспорт | подтверждение регистрации; если нет ответа — ходатайство в суд об истребовании |
| **Банки (выписки)** | Движение денег за 3 года + период процедуры | Источники поступлений, зачисление кредитов (дата возникновения долга!), крупные переводы физлицам, снятия, взыскания ФССП, остаток на дату |
| **Должник** (запрос ФУ) | Пояснения, документы, карты | причина неплатёжеспособности («снижение доходов» / «расходы на детей»), пояснения по активам |

Также в папке: решение суда, определения о включении требований, ЕФРСБ-сообщения (bankruptcy_message_*.pdf), скрин «ОБЪЯВЛЕНИЯ О НЕСОСТОЯТЕЛЬНОСТИ» (Коммерсантъ), свидетельства (смерть супруга, брак), справки МСЭ, договоры аренды.

## Чек-лист извлекаемых переменных

### Дело
- № дела (формат А18-XXXX/YYYY и т.п.), суд (полное название + адрес), судья (если известен)
- Дата решения о признании банкротом (= дата утверждения ФУ; сверить между решением, ЕФРСБ и Коммерсантом — бывают расхождения, приоритет у текста решения)
- Срок процедуры: с даты решения + 6 мес.; продления (даты из определений)
- Дата заседания по рассмотрению отчёта

### Должник (все формы!)
- ФИО: именительный, родительный, дательный падежи; род (для «признан/признана», «действовал/действовала»)
- Дата и место рождения, ИНН, СНИЛС, адрес регистрации (+ с какой даты, по МВД), паспорт
- Статус: пенсионер (вид пенсии, размер) / работает (работодатель, 2-НДФЛ по годам) / ИП (ОГРНИП, период) / безработный; инвалидность (группа, справка МСЭ)
- Причина неплатёжеспособности (со слов должника): «снижение доходов» или «увеличившиеся расходы на содержание несовершеннолетних детей»

### Семья
- Брак: с кем, дата, № записи; расторжение (дата прекращения, решение суда); смерть супруга (дата, № записи, свидетельство)
- Дети: ФИО, даты рождения, кто несовершеннолетний НА ДАТУ ПАКЕТА (пересчитывать!), инвалидность детей
- Формулы для отчёта: «Должник в браке не состоит / состоит в браке» + «На иждивении имеет/не имеет несовершеннолетних детей» — ДОЛЖНЫ соответствовать ЗАГС и СФР (пособия на детей = иждивенцы есть!)

### Кредиторы и реестр
- По каждому: название, адрес; суммы с разбивкой — основной долг / просроченные проценты / неустойка-штрафы-пени / госпошлина; очередь (налоги-недоимка может быть 2-й очередью!); дата и № определения о включении
- Итог реестра = сумма всех; дата закрытия реестра (= дата решения + 2 мес. по практике)
- Процент удовлетворения (обычно 0%)

### Публикации
- Коммерсантъ: № объявления (77...), № газеты, дата, страница
- ЕФРСБ: № сообщения, дата (о признании банкротом; отдельно — о результатах инвентаризации, о заключении, о собрании)
- Дата уведомления кредиторов (почтовые квитанции)

### Имущество и сделки
- Каждый объект: тип, идентификатор (кадастровый №/VIN), период владения, основание приобретения, дата и способ выбытия, документ-закрытие (см. дерево решений)
- Опись: дата (= дата закрытия реестра, как правило), «имущество не выявлено» или перечень

### Счета
- Полный список из ответа ФНС: банк, № счёта, статус (основной / заблокирован / закрыт ранее), остаток
- Основной счёт: №, банк, обороты за процедуру (приход/расход/остаток на дату выписки)

### Деньги
- Источники поступлений за 3 года; крупные операции, требующие квалификации; исследуемый период Заключения = [дата подачи заявления − 3 года; дата подачи заявления]
