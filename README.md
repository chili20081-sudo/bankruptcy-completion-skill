# bankruptcy-completion — навык для завершения процедур банкротства физлиц

Навык (skill) для [Claude Code](https://claude.com/claude-code), который по папке с ответами на запросы по должнику **самостоятельно** составляет пакет документов для подачи в арбитражный суд на завершение процедуры реализации имущества гражданина (банкротство физического лица, ФЗ-127 «О несостоятельности (банкротстве)»).

Выведен из анализа **200+ реальных завершённых дел** практики действующего арбитражного управляющего: изучены итоговые документы, ответы регорганов и логика решений по всем встречающимся сценариям.

## Что умеет

Вы даёте папку с документами по должнику (ответы ГАИ, ЕГРН, ЗАГС, ФНС, СФР, МЧС, Росавиации, выписки банков, судебные акты) — навык сам:

1. **Читает все документы** и извлекает данные (должник, семья, доходы, имущество, сделки, кредиторы, публикации);
2. **Определяет сценарий** по дереву решений: есть ли имущество (иммунитет единственного жилья / авто при инвалидности), сделки в 3-летнем периоде подозрительности (оспаривать по ст. 61.2 или обосновать отказ — 5 типовых обоснований), «фантомные» активы (ДКП, акт сдачи на металлолом), супруг (жив / умер / разведён / банкротится параллельно), осложнения (ходатайство кредитора о неосвобождении, собрание кредиторов, требование реструктуризации);
3. **Составляет пакет из 6 документов**: ходатайство о завершении (ст. 213.28), отчёт финансового управляющего, отчёт об использовании денежных средств, опись имущества, заключение о признаках фиктивного/преднамеренного банкротства (ПП РФ № 855), реестр требований кредиторов — плюс сателлитные документы по ситуации (продление срока, перечисление вознаграждения с депозита, отзыв на неосвобождение, пояснения суду, согласие на расходные операции и др.);
4. **Проверяет пакет** по чек-листу из 6 разделов (сквозная согласованность номеров дел, ФИО, дат, сумм — включая типовые ошибки копипасты, найденные в реальных делах).

## Установка

1. Скопируйте папку репозитория в директорию навыков Claude Code:
   - Windows: `C:\Users\<вы>\.claude\skills\bankruptcy-completion\`
   - macOS/Linux: `~/.claude/skills/bankruptcy-completion/`
2. **Заполните свои данные** в `reference/05-реквизиты-фу.md` (ФИО, СРО, страховка, реквизиты — вместо `{{плейсхолдеров}}`).
3. По желанию: подготовьте папку со своими эталонными делами (готовые .doc прошлых завершений) — навык будет использовать их как шаблоны форматирования через MS Word.
4. В Claude Code скажите: «собери завершение по [должник], папка такая-то».

## Важные оговорки

- **Не юридическая консультация.** Навык — инструмент помощника; итоговую ответственность за документы несёт арбитражный управляющий. Проверяйте каждый пакет перед подписанием и подачей.
- Практика выведена из дел арбитражных судов Северо-Кавказского региона (основной массив), а также Московской области и г. Москвы; формулировки типовые по ФЗ-127 и применимы шире, но региональные особенности проверяйте сами.
- Навык не подаёт документы и не выдумывает данные: каждая цифра должна иметь источник в папке дела.
- Судебная практика меняется (например, Обзор Президиума ВС РФ от 18.06.2025 по банкротству граждан уже учтён) — следите за актуальностью правовых позиций.
